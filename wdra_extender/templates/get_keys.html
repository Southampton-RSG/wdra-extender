Input Twitter API Keys

To launch searches via the Twitter API we require several authentication tokens. These tokens are obtained by signing up for a developer/researcher account here.

The Twitter API comes with several 'endpoints' currently available are the 'search_tweets' and the 'search_archive'.

'search_tweets' is the most common method which uses the standard product track. It can retrieve tweets from the last 7 days.

'search_archive' is the the endpoint associated with the academic research product track. It can retrieve tweets from the entire archive.
