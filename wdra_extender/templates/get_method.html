Search Tweets

1. Select Tweet Return Method

There are two search methods: Search and ID:
  • Search has two forms:
    • The basic form can:
      • Include and exclude terms
      • Set the total number of tweets to return
      The basic form will return the tweet ID and the tweet text as is the default for the APIv2, it is also limited to the last 7 days.
    • The advanced form can:
      • Include and exclude terms
      • Set the total number of tweets to return with batch size control
      • Allow selection of date and time range and/or selection of tweet ID range (within the limitation of your API credentials, non-academic users are still limited to the last 7 days)
      • Allow selection of the return information giving access to the full suit of API return fields
  • ID is a tweet hydrating feature. By supplying a list of tweet IDs the API is used to obtain all available fields from the tweet.
